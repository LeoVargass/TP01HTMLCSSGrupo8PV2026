-Descripción

Este proyecto consiste en una plataforma web estática para la gestión de tutorías entre estudiantes.
Se desarrolló utilizando HTML para la estructura y CSS para los estilos, junto con Git y GitHub para el trabajo colaborativo.

-Páginas
index.html → Página principal con estadísticas y novedades del sistema.
proyectos.html → Listado de tutorías con filtros y acceso a cada proyecto.
detalle.html → Información completa de cada tutoría, incluyendo descripción, recursos y equipo.
perfil.html → Perfil de usuario con datos personales y rol.
- Tecnologías
HTML5
Git / GitHub

Las páginas están conectadas entre sí mediante enlaces, permitiendo recorrer todo el sitio.
